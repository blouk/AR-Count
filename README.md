# Count number of markers

## Get the code
````bash
git clone https://github.com/blouk/AR-Count.git
cd ar_count
````
## Install dependencies
````bash
 npm install
````
You need npm installed : https://docs.npmjs.com/getting-started/installing-node

## Dev build
````bash
 npm start
````

## Build
````bash
 npm run build
````
